# 统计页(Stats) 功能升级计划

## 1. 状态管理与导航
- **新增视图模式切换**：在顶部导航栏下方增加 "月统计" / "年统计" 的切换开关。
- **新增日期选择器**：
  - 替换原有的依赖 `sharedCurrentYM` 的逻辑，使用本地 `currentDate` 状态。
  - **月模式**：显示 "YYYY年MM月"，支持左右切换月份。
  - **年模式**：显示 "YYYY年"，支持左右切换年份。

## 2. 数据处理逻辑
- **月度数据**：保持现有逻辑，但数据源根据新的 `currentDate` 进行过滤。
- **年度数据**：
  - **折线图数据**：按月份（1-12月）聚合当年所有记录，计算每月的收入和支出总额。
  - **排行榜数据**：按分类聚合当年所有记录，计算每个分类的总金额，并按降序排列。同时计算各分类在全年总支出/收入中的占比。

## 3. UI 界面开发
- **月统计视图**：保留现有的饼图和明细列表布局。
- **年统计视图**：
  - **收支趋势图**：使用 ECharts 绘制双折线图（收入 vs 支出），展示全年资金流向趋势。
  - **分类排行榜**：
    - 展示分类名称、图标、总金额。
    - 增加进度条背景展示占比。
    - 列表按金额降序排列。

## 4. 技术实现细节
- 使用原生 `Date` 对象处理日期的加减和格式化（因项目未引入 dayjs）。
- 复用 `useLedger` 中的 `records` 数据源。
- 复用 `echarts` 组件库来实现新的折线图。
